# How to build from scratch

#### Install python and install git
https://www.python.org/<br />
https://git-scm.com/install/
#### Clone repository into an editor of choice
- I personally used VSCODE

#### Install the required libraries
- Run: <br />
```pip install -r requirements.txt```

#### Compile
- Run: <br />
```py -m PyInstaller .\TomoKoreFacepaintTool.spec```

#### Enjoy
- The compiled file for usage will be in ```/dist```
